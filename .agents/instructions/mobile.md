# Mobile Instructions

- Точка входа для mobile-инструкций: [Project-specific](./mobile/project_specific.md)
- Порядок чтения: `project_specific.md` -> platform guideline -> тематические документы

## Приоритетность

- При конфликте правил для mobile-кода действует порядок:
  `project_specific.md` -> platform guideline -> тематические mobile-доки -> `AGENTS.md`.
- Пример: по комментариям в mobile действует правило из mobile-гайда, даже если общие инструкции допускают doc-комменты.
