# Resources (KMP)

- Запрещены строковые литералы, хардкод иконок/изображений и другие запахи кода
- Ресурсы хранить только в `composeResources/`
- Обращаться к ресурсам только через сгенерированный `Res` (`Res.string.*`, `Res.drawable.*`)
